# 💻 技术栈规范 (Tech Stack)

### 1. 基础框架 (Framework)
* **Next.js (App Router)**: 用于构建高性能的 React 网页应用。
* **TypeScript**: 提升代码严谨性，确保装备参数字段的类型安全。

### 2. 样式与 UI (Styling)
* **Tailwind CSS**: 采用原子化 CSS 快速构建户外硬核视觉风格。
* **Lucide React**: 统一的极简图标库。
* **Framer Motion**: (可选) 用于实现置物架展开或卡片切换的丝滑动画。

### 3. 数据处理 (Data & State)
* **LocalStorage**: 纯前端阶段用于持久化用户的个人装备库数据。
* **Zustand / React Context**: 用于管理全局状态（如当前选择的赛事、用户装备列表）。

### 4. 开发范式 (Vibe Coding Principles)
* **Prompt-First**: 所有的 UI 调整与逻辑增加优先通过自然语言描述。
* **Component-Based**: 保持组件高度模块化，方便 AI 快速读取和修改特定部分。
