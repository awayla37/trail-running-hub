# Sprint 01（MVP Build Sprint）

---

## 🎯 Sprint Goal（冲刺目标）

Build a working MVP of Trail Hub:

User can select a route, input gear, and get recommendation.

完成 Trail Hub 的最小可运行版本：

用户可以选择路线 → 输入装备 → 获取推荐

---

## ⏱ Duration（周期）

7 Days

---

## 📦 Scope（范围）

### ✅ Must Have（必须完成）

1. Project setup（项目初始化）

* Next.js project
* Basic folder structure

---

2. Route List Page（路线列表页）

* Display routes from routes.json
* Click into route detail

---

3. Gear Input（装备输入）

* Input text (brand + model)
* Match gear from gear.json

---

4. Recommendation Engine（推荐逻辑）

* Simple scoring based on terrain
* Return ranked gear

---

5. Result Display（结果展示）

* Show best gear
* Show score / reason

---

## ⚠️ Not in Scope（不做）

* User system
* Database
* AI API
* GPX upload

---

## 🧠 Key Deliverable（关键产出）

A live demo on Vercel:

* Shareable link
* Usable by others

一个可以分享的在线产品版本
