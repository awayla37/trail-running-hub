# Trail Hub Engine 🏔️⚡

**RPG-Inspired Tactical Loadout & Synergy Engine for Trail Runners.**

Trail Hub 不仅仅是一个越野跑记录工具，它是越野跑者的“战备工作台”。我们通过精密的数据演算，解决越野跑核心痛点：**环境与装备的精密匹配。**

### 核心理念：从工具到社群 (Tool to Network)
- **精密配装 (RPG Workbench)：** 像在 RPG 游戏中切换武器一样配装。锁定赛道，实时预览装备协同指数。
- **数据驱动 (Field Intel)：** 告别感性的“种草”，回归硬核的面料科技与实战战损报告。
- **战术名片 (Digital Card)：** 生成高调性的实战归档卡片，实现跨平台的专业影响力扩散。

### 核心功能模块
- **[Routes] 赛事编年史**：集成全球赛道 GPX 矩阵，包含海拔剖面、路况分布及动态气象看板。
- **[Gear Lab] 装备实验室**：涵盖 Footwear (跑鞋)、Apparel (服装科技)、Pack (背包挂载) 的多维参数库。
- **[Synergy HUD] 实时配装工作台**：切换装备时，“协同作战指数”实时演算跳动。
- **[Armory] 个人军械库**：存储跑者战绩、数字资产及带有结构化标签的避坑报告。

### 技术栈
- **Frontend:** Next.js (App Router), Tailwind CSS, TypeScript.
- **Engine:** Custom Haversine GPX Parser, Synergy Logic Controller.
- **Design:** Hardcore Outdoor Aesthetic (Earth & Sandstone / Pine Green).
