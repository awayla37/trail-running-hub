# 评分引擎逻辑说明 (v1.0)

## 百分制计算公式
总分 (100) = 抓地力(40) + 保护性(30) + 缓震感(20) + 轻量化(10)

## 加分规则示例
1. **抓地力 (40分)**:
   - 若赛道 `mud` > 20% 且 鞋子 `lug_depth_mm` >= 5: +20分。
   - 若赛道 `technical_rock` > 30% 且 `outsole_tech` 包含 "Vibram": +20分。

2. **保护性 (30分)**:
   - 若赛道 `technical_rock` > 30% 且 `rock_plate` 为 true: +15分。
   - 若 `stack_height` > 30mm: +15分。

3. **轻量化 (10分)**:
   - 若 `weight_g` < 250g: 满分 10分。
   - 每增加 20g 扣 1 分。
