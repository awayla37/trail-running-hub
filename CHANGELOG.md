# 更新日志 (Change Log)

## [1.2.0] - 2026-03-15
### 🛰️ 实时气象感知 (The Weather Update)
- **API 深度集成**：成功接入 Open-Meteo 实时气象接口，根据赛道经纬度抓取真实降水与湿度数据。
- **自动化逻辑联动**：实现了“气象 -> 赛道湿度 (Wetness) -> 装备评分”的全自动传导，彻底告别假数据模拟。
- **状态反馈系统**：新增 Weather Loading 状态，模拟卫星同步的视觉效果，增强工具的专业感。

### 🛠️ 健壮性改进
- **空值防御**：重构评分逻辑，增加了对 Race 和 Gear 对象的空值校验，彻底修复了对比模式下的红字报错。
- **布局自适应**：VS 模式现在支持单装备/双装备动态切换布局。
